Skype fue desarrolldo en sus inicios con QT, biblioteca multiplataforma para desarrollar aplicaciones utilizando el lenguaje C++ pero a los dos meses de salida la primera versión se adopto el uso de Delphi, un producto tipo RAD(Rapid Aplication Development) originalemte desarrollado por Borland y actualmente por Embarcadero, para que pueda ser utilizado con sistema operativo Windows aunque las ultimas versiones tambien generan ejecutables para MacOs, Lunix y .NET.

Las versiones móviles están escritas en el idioma nativo compatible con esas plataformas, por lo que el cliente de iOS está escrito en Objective C, Android en Java, Windows Phone en C #. Las versiones móviles utilizan bibliotecas C ++ similares, pero optimizadas para dispositivos móviles, en el escritorio para las funciones básicas de llamadas. Las otras plataformas (Mac, XBox, Sony Smart TV, etc.) usan el lenguaje de desarrollo nativo para la plataforma dada y bibliotecas C ++ similares.
